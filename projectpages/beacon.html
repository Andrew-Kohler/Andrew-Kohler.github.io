About the Project

Project Beacon is the working title for my undergraduate honors thesis at the University of Florida. As a project-based thesis, its focus is on demonstrating the programming and game development skills that I've built over my time as a Digital Arts and Sciences (BS) student.

As a game, the project is about two friends going for a hike in the woods and having meaningful conversations about how one of them is dying. In this way, it pushes an envelope that is rarely able to be explored in my degree program - storytelling. I aim to make it impress narratively with the story and technically with its visuals and gameplay; I'm implementing a unique 2.5D one-bit aesthetic and two-character gameplay that have already seen me write a complex shader for shadowcasting and implement Unity's NavMesh systems.

Programming Accomplishments

  • Developing a movement system that allows different player characters to become an autonomous agent or under player control on a button press
  • Working with Unity's NavMesh systems to create an intelligent follower character AI
  • Using Unity's shader language to create shadow and water shaders that bolster a one-bit aesthetic
  • Implementing a dialogue system that positions characters in real time to create live cutscenes

Other Accomplishments

  • Designing and creating all art assets, narrative content, and gameplay features
