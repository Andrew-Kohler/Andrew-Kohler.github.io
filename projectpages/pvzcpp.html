Content-Driven Engine

Planting Roots with JSON

I developed a content-driven game engine over the course of a semester; the end product allows for usage of a custom JSON scripting format to create content from atomic modules called Actions, in a structure broadly similar to major visual scripting systems, built only from C++.

This functionality is supported by a host of custom data containers, singleton managers, construction processes, and unit tests that ensure each module is working in a thorough fashion. My favorite piece to write was the Attributed structure, which enables the injection of instance attributes via JSON at runtime into objects.

Event System

Brain-To-Brain Instant Communication

As everyone on our team had worked to construct similar engines, we made a decision as a group on which was the best to use. This meant I was working in a familiar but different codebase to my own engine. I was able to familiarize myself with my new environment by helping to develop a missing component of this new world: an event system.

I created an EventService active during the program's runtime capable of listening for Events and executing callback functions (Reactions) on specific object instances. I also integrated this system with our JSON-based content system, wrapping events and reactions in an easy-to-use JSON format (pictured left).

Zombie Wave Management & Spawning

Zombies On Your JSON

My major contribution to our recreation of Plants Vs. Zombies was the wave spawning system. The original game's system has no published documentation; I made an accurate recreation in our JSON action scripting language through research and experimentation, breaking it down into its most atomic pieces and creating the building blocks I needed.

The final product has all of the same critical configuration parameters as the original game, allowing for a rapid and game-accurate system to use in level creation and balancing.

Other Contributions

Grazing The Roof, And Everywhere Else

In addition to my development of the game's wave manager and overall state management, I assisted with polish in almost every area, helping to create plants, finalize economy integration, and add visual and sound effects.
