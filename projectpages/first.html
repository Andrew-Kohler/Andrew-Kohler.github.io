About the Project

Race The Stars is the first video game I ever made! It remains on this website as a reminder of where I began.

It's a 2D platformer made in Scratch for the 2019 Codecraft Innovation Games. It has 2 levels, my first ever sprites, and buggy collision as a result of using color-based collision. I fought past a lot of Scratch's little quirks to make things like the scrolling level work!

Programming Accomplishments

  • Built a 2D sidescrolling platformer system within Scratch that loads and deloads assets based on screen position
  • Built unique level mechanics and hazards for each level

Other Accomplishments

  • Created all spritework for the game, as well as the title theme
  • Designed levels to tell a story of running through the night to see the sunrise
