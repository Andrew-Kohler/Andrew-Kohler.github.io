About the Project

It Bears Repeating is a 2D platforming beat 'em up centered around an unassuming bear becoming entangled in a Jamed Bond-esque super spy time loop, all while just being a bear of average bear intelligence and skills. It was the first large project I programmed in Unity, and through it I learned about Unity's UI systems, 2D parallax scrolling, and Cinemachine, and figured out on my own how to program the more advanced player and enemy movement and combat mechanics.

Programming Accomplishments

  • Programmed a 2D platformer character controller with beat-em-up elements (high knockback dash, damage-reducing crouch, 3 hit combo, health and stamina)
  • Created enemies capable of navigating over TileMap terrain to attack the player
  • Created various interactable elements such as healing pickups, exploding barrels, and destructible barriers
  • Used Cinemachine and original scripting to create unique cutscene moments

Other Accomplishments

  • Created all spritework and animation for the game, including all bear animations
  • Responsible for all game design decisions and design implementation (level building, story, etc.)
